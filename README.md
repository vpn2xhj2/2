Shadowsocks软件介绍
SS，全称为 Shadowsocks（下文简称为SS），是一种基于 Socks5 代理方式的网络数据加密传输包的工具，可以使用其于科学上网 也就是翻墙 ，是目前最稳定，最常用的科学上网工具之一，全平台支持

SSR，全称为 ShadowsocksR（下文简称为SSR） 来历：

ShadowsocksR是breakwa11发起的Shadowsocks分支，在Shadowsocks的基础上增加了一些数据混淆方式，称修复了部分安全问题并可以提高QoS优先级。后来贡献者Librehat也为Shadowsocks补上了一些此类特性，甚至增加了类似Tor的可插拔传输层功能。

ShadowsocksR开始时曾有过违反GPL、发放二进制时不发放源码的争议，不过后来ShadowsocksR项目由breakwa11转为了与Shadowsocks相同的GPL、Apache许可证、MIT许可证等多重自由软件许可协议。

2017年7月19日，ShadowsocksR作者breakwa11在Telegram频道ShadowsocksR news里转发了深圳市启用SS协议检测并被大量用户转发，引发恐慌。7月24日，breakwa11发布了闭源的SS被动检测程序，引发争议。 2017年7月27日，breakwa11遭到自称 “ESU.TV” 的不明身份人士人身攻击，对方宣称如果不停止开发并阻止用户讨论此事件将发布更多包含个人隐私的资料，随后breakwa11表示遭到对方人肉搜索并公开个人资料的是无关人士，为了防止对方继续伤害无关人士，breakwa11将删除GitHub上的所有代码、解散相关交流群组，停止ShadowsocksR项目。但项目已被多人fork。并有人在其基础上继续发布新的版本，例如SSRR

小火箭（Shadowrocket）软件使用指南
小火箭（Shadowrocket）是一款功能强大的代理工具，旨在帮助用户科学上网，突破网络限制。无论是在iOS设备上使用，还是在电脑上配置，掌握小火箭的安装与使用方法都至关重要。以下是关于小火箭的全面指南，包括安装、配置及使用。

一、安装小火箭
iOS设备安装：

App Store下载：打开你的iOS设备，前往App Store，搜索“小火箭”或“Shadowrocket”，点击下载。如果你的Apple ID是非中国区的，可以直接进行下载。
账户设置：若在中国区App Store中找不到，建议使用其他国家的Apple ID进行下载，或通过朋友的账号获取。
电脑安装：

Mac电脑：小火箭在Mac上并不直接支持，但可以使用类似的工具如V2Ray或Shadowsocks。下载安装相应的客户端并进行配置。
Windows电脑：下载V2Ray或Shadowsocks的Windows版本，按官网说明进行安装和设置。
二、配置小火箭
打开应用：安装完成后，打开小火箭应用。
添加节点：
在首页点击“配置”，选择“添加配置”。
输入你的代理节点信息，包括服务器地址、端口、加密方式等。
如果你有节点订阅链接，可以直接导入链接，简化配置过程。
选择连接模式：根据需求选择“全局模式”或“规则模式”。全局模式下所有流量都通过代理，而规则模式则仅针对特定流量。
三、使用小火箭
连接代理：在配置页面选择你添加的节点，点击连接按钮，待连接成功后，状态会显示为“已连接”。
流量监控：小火箭内置流量监控功能，可以实时查看使用情况，帮助你合理控制流量。
速度测试：使用测速功能，测试不同节点的速度，选择最优节点进行连接。
四、编程与高级功能
小火箭不仅是简单的代理工具，也为程序员提供了丰富的配置选项。

自定义配置：对于有编程基础的用户，可以自定义规则，根据不同需求调整代理设置。
脚本支持：小火箭支持多种脚本，可以通过编程实现更灵活的代理策略。
API调用：在某些场景下，可以通过API与小火箭进行交互，实现自动化管理。
五、常见问题及解决方案
连接失败：检查节点信息是否正确，确保网络连接正常。
速度缓慢：尝试切换到其他节点，或者更改连接模式，选择速度更快的配置。
隐私问题：在选择代理节点时，优先选择信誉良好的服务商，以保护个人隐私。
